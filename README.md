# ✈️ Airline Price Analysis Dashboard

## Project Overview
This project analyzes airline ticket prices using Power BI.

## Tools Used
- Power BI
- Power Query
- DAX
- Excel

## Key KPIs
- Average Ticket Price
- Average Flight Duration
- Total Flights
- Price by Airline
- Price by Source City
- Price by Destination
- Price by Class

## Dashboard Features
- Interactive slicers
- Dynamic charts
- KPI cards
- Trend analysis
