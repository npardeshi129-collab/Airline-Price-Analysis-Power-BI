# ✈️ Airline Price Analysis Dashboard

An interactive Power BI dashboard that analyzes airline ticket prices using data visualization, DAX, and Power Query.
This project presents an interactive Power BI dashboard that analyzes airline ticket prices across different airlines, travel classes, source cities, and destination cities. It transforms raw flight data into meaningful business insights using Power Query, DAX, and interactive visualizations, helping users explore pricing trends and compare travel options effectively.
